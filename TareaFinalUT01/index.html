Explicación de la Aplicación

Funcionamiento de la Aplicación

[ Consta de tres páginas principales ]


1. Página de Presentación

En esta página inicial se presenta el propósito de la aplicación, explicando su funcionalidad y cómo puede ser útil para los usuarios. Aquí, los visitantes encontrarán un botón que les llevará a la siguiente página para iniciar el proceso de registro

2. Página de Introducción de Datos

Esta página permite a los usuarios registrarse introduciendo su nombre y dirección de correo electrónico en un formulario sencillo. Una vez completado el formulario, los datos se envían a la aplicación, donde se almacenan de forma segura. Tras el envío, el usuario es redirigido automáticamente a la siguiente página para visualizar la lista de registros

3. Página de Visualización de Datos

EEn esta página, los usuarios pueden ver una lista de todos los registros almacenados, que incluye los nombres y direcciones de correo electrónico de aquellos que se han dado de alta. Esta funcionalidad permite a los administradores o a los propios usuarios tener un acceso rápido y claro a la información recopilada
